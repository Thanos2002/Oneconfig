# Security Policy

## Supported Versions

Currently, only the latest release of OneConfig is supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| > 1.0.0 | :white_check_mark: |
| < 1.0.0 | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within OneConfig, please do not open a public issue.

Instead, please send an email to the maintainers (or create a draft security advisory if enabled on this repository). We will address all security vulnerabilities promptly.

### What to include

- A detailed description of the vulnerability.
- Steps to reproduce the issue.
- Potential impact and an idea of how to fix it (if you have one).

Thank you for helping keep OneConfig safe!
